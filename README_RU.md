## Установка

Чтобы использовать этот проект, следуйте этим шагам:

1. **Клонируйте репозиторий:**
   Убедитесь, что Git установлен на вашей системе.
   ```bash
   git clone https://github.com/GogaNotClown/GogaNotClownProfile.git
   cd GogaNotClownProfile
   ```

2. **Скачивание пакетов NPM:**
   Убедитесь, что Node.js установлен на вашей системе.
   ```bash
   npm install
   ```

3. **Запуск проекта:**
   Эта команда запустит проект на localhost.
   ```bash
   npm run dev
   ```
